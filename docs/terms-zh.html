2. 服务说明

"胖了吗"提供正念身体追踪体验。应用允许您：

  • 记录每日身体感觉和感受
  • 追踪体重测量（可选）
  • 记录心情标签和个人笔记
  • 查看随时间变化的洞察和趋势
  • 与 Apple 健康应用集成（可选）
  • 在设备上本地存储数据（v1.0）

应用仅用于个人、非商业用途。

未来功能（v1.1+）：iCloud 同步、Face ID/Touch ID 应用锁、数据导出。
